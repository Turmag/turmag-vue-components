# turmag-vue-components

Библиотека компонентов на Vue

# Installation
Install [turmag-vue-components](https://www.npmjs.com/package/turmag-vue-components).

```
npm i turmag-vue-components

# Or run this to use yarn
yarn add turmag-vue-components

# Or run this to use pnpm
pnpm add turmag-vue-components
```

# Usage
Import components in your project then use it

```
import { UiFlex, UiText } from 'turmag-vue-components';
```
